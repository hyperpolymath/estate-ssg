# Welcome

This is **Forth Estate**, the stack-based static site generator.

## Features

- Pure Forth implementation
- YAML frontmatter
- Markdown parsing
- Template substitution

## Getting Started

```forth
\ Load the SSG
gforth src/forth-estate.fs

\ Run tests
test-full
```

Built with the concatenative paradigm.
